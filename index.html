Money Jar

This is my Money Jar web Application. It was my Senior Project. It is a virtual jar, Users can create a virtual Jar on their account and invite others to join it. Total value of the jar can be increased as desired by all participants.

Money jar is built using PHP, HTML, CSS, and Bootstrap. It was built using a PostgresSQL database and in VsCode

Keep in mind, this page is meant to be veiwed via mobile only. Money Jar was a solo project, built in about 40 hours over the course of 2 months. I hope to improve it as I get the time. Fix the many bugs, and work with a designer to make it look better.

DnD Initative Tracker

This is my Initiative Tracker application. I built it during a web engineering class at BYU-I. We were required to switch tasks, but with more time I would love to link it to a database, and fix the bugs.

The tracker allows users to submit characters to a database, and then retrieve them later. A user will use the Roster system to add players for a game encounter and the system will order and track this game

This project was also a solo project, built over about a month. It uses the same systems as Money Jar, using PHP, HTML, CSS, and Bootstrap. It was built using a PostgresSQL database and in VsCode
